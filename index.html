Introduction

What is OpossumProse?

OpossumProse is a native macOS desktop application for long-form writing. It supports three distinct project types — Screenplays, Novels, and Stage Plays — each with a format-specific editor, a shared suite of creative tools, and an AI co-writer powered by Google Gemini.

Projects are stored locally on disk. Everything auto-saves. There are no word processors, no cloud sync requirements, and no subscription tiers — just a clean environment for writing.

Three project types

  • Screenplay (film & TV)
  • Novel (chapters + manuscript view)
  • Stage Play (theatre / live performance)

Available on

  • macOS (Apple Silicon & Intel)
  • Distributed as a .dmg installer
  • Built with Electron + React
Powered by Gemini AI features require a Google Gemini API key. Keys are stored locally and never leave your machine. See AI Features for setup details.
Setup

Getting Started

Download, install, and write your first scene in under two minutes.

Installation

  1. Download the latest .dmg from the Releases page.
  2. Open the .dmg and drag OpossumProse.app into your Applications folder.
  3. Launch the app. macOS may ask you to confirm opening an app from an unidentified developer — click Open.
  4. You'll land on the Dashboard. Create your first project.

API Key Setup

To use AI features you need a free Google Gemini API key.

  1. Visit aistudio.google.com and generate an API key.
  2. In OpossumProse, open Settings and paste your key into the Gemini API Key field.
  3. The key is stored locally in your app data — it is never uploaded anywhere.
Tip The Gemini Flash model is used by default — it is fast and free-tier eligible. No billing required for casual use.
Navigation

Dashboard & Projects

The Dashboard is the home screen of OpossumProse. From here you create new projects and open recent ones.

Creating a project

  1. Click New Project on the Dashboard.
  2. Choose a project type: Novel, Screenplay, or Stage Play.
  3. Enter a project name. A directory is created on disk at the chosen location.
  4. The editor opens immediately and auto-saves from that point on.

Writing Mode

Screenwriting

The Screenplay editor enforces industry-standard formatting automatically — Courier New at 12pt, correct margins, and proper element positioning — so your output is submission-ready without configuration.

Script elements

Every line in the editor has an element type. The element controls its formatting, indentation, and capitalization automatically.

ElementShortcut / Smart behaviorFormatting
Scene Heading Tab cycles to Action All caps · bold · left-aligned · example: INT. KITCHEN - DAY
Action Enter creates another Action; Tab → Character Normal case · left-aligned · full page width
Character Enter → Dialogue; Tab → Dialogue All caps · centered at 2.2 in from left
Parenthetical Tab → Dialogue; Enter → Dialogue Normal case · indented 1.6 in · 2 in wide
Dialogue Enter → Character; Tab → Character Normal case · indented 1 in · right margin 1.5 in
Transition Tab → Scene Heading; Enter → Scene Heading All caps · right-aligned · example: CUT TO:
Tab cycling Pressing Tab on an empty line cycles to the next logical element type. This is the fastest way to navigate the format without a mouse.

Character autocomplete

When typing on a Character line, OpossumProse shows an autocomplete dropdown of character names already in your script. Use arrow keys to navigate and Enter or Tab to accept.

Scene navigation

The left sidebar lists every scene heading in order. Click any scene to jump directly to it. Scene headings are numbered automatically.

Writing Mode

Novel Writing

The Novel editor organizes your manuscript into chapters. Each chapter is a titled block of prose. Chapters can be reordered, renamed, and navigated from a collapsible sidebar.

Chapters

  • →Add chapters with the + Add Chapter button in the sidebar.
  • →Rename any chapter by editing its title inline in the sidebar or at the top of the editor.
  • →Reorder chapters using the up/down arrows next to each chapter name.
  • →Delete chapters with the trash icon (a confirmation is shown).
  • →Word count is tracked per chapter and shown in the status bar.
  • →Total word count across all chapters is displayed in the header.

Manuscript view

Toggle Manuscript view to display all chapters in a single scrollable document — useful for reading passes and continuity checks. Click ← Chapter view to return to single-chapter editing.

Auto-save

Changes are debounced and saved to disk automatically after 1.5 seconds of inactivity. A subtle indicator shows when a save is in progress. No manual save required.

AI in the novel editor

The AI Writer panel (see AI Writer) works in novel-chapter mode. When generating, it passes the end of the previous chapter as context so the generated text flows naturally from what came before.

Writing Mode

Theatre & Stage Play

The Stage Play editor shares the same element-based engine as the Screenplay editor, but with vocabulary and placeholders adapted for live theatre — acts instead of scenes, stage directions instead of action lines, and BLACKOUT instead of CUT TO.

Stage play elements

ElementTheatre placeholderNotes
Scene Heading ACT ONE Used for act and scene headings; all caps.
Action Stage direction... Stage directions — lighting, movement, props, set notes.
Character CHARACTER NAME Same as screenplay — uppercase, centered.
Parenthetical (quietly) Delivery notes in parentheses below the character name.
Dialogue Dialogue... The character's spoken lines.
Transition BLACKOUT. Lighting or scene transitions — right-aligned, all caps.
Same keyboard flow Tab cycling, Enter smart-advance, and character autocomplete all work identically to the Screenplay editor.

Title page & export settings

Stage play projects include the same title page editor as screenplays. The default export settings enable the title page and allow toggling scene numbers and revision watermarks. Export formats are identical: PDF, DOCX, and Fountain.

AI Writer in stage mode

The AI Writer supports a screenplay-act mode which is used for stage plays. Prompts describe the major beats, character arcs, and central conflict of an act, and the AI generates a fully-formatted script segment with stage directions and dialogue.

AI Features

AI Overview

OpossumProse integrates Google Gemini to provide three distinct AI capabilities: an inline generator that writes formatted content from a prompt, a conversational chat assistant, and an automated continuity scanner.

AI Writer

Describe a scene, act, or chapter beat. The AI generates a fully-formatted draft and streams it into a preview panel. Insert it at your cursor or discard and try again.

AI Chat

A multi-turn conversation assistant that has access to your script as context. Ask it to develop a character arc, punch up dialogue, or brainstorm alternatives.

Continuity Checker

Sends your script to Gemini for a full read-through. Returns tagged findings organized by category: character name inconsistencies, timeline anomalies, location naming issues, and more.

API key required All AI features require a Gemini API key configured in Settings. Responses stream in real-time — a stable internet connection is required.
AI Features

AI Writer

AI Writer generates formatted script or prose content from a natural-language prompt. It is context-aware — by default it includes the surrounding text so the generated content flows from what you've already written.

Modes

ModeTriggered fromPrompt guidance
Write Scene with AI Screenplay & Stage Play editor Where are we, who's there, what happens, what's the tension or turn…
Write Act with AI Screenplay & Stage Play editor Major beats, character arcs, how it opens and closes, central conflict…
Write Chapter with AI Novel editor Setting, conflict, what changes for the characters…

How it works

  1. Click the Write with AI button (wand icon) in the editor toolbar.
  2. Type your prompt describing what should happen.
  3. Optionally toggle Use surrounding context off if you want a standalone piece.
  4. Click Generate. The response streams into the preview pane.
  5. Review the result. Click Insert to place it at your cursor, or Regenerate to try a new take.
Context toggle When context is on, the AI sees the last ~1500 characters of the previous chapter/scene and your current position. This dramatically improves voice and continuity consistency.
AI Features

AI Chat

AI Chat is a persistent side panel conversation with Gemini. On first message it sends your current script as context, giving the AI a full picture of your project before any questions are asked.

Usage

  1. Click the Chat icon in the toolbar to open the side panel.
  2. Type your question or instruction. Press Enter to send, Shift+Enter for a new line.
  3. The response streams in character by character.
  4. Continue the conversation — the AI remembers everything said in the session.
  5. Close the panel to end the session. History is not persisted between sessions.

Things to ask

  • 💡"What are some alternative endings for the second act?"
  • 💡"The dialogue in scene 14 feels stilted. How would you punch it up?"
  • 💡"What's the emotional arc of JAMES across the script?"
  • 💡"Suggest three ways to raise the stakes in the third act."
  • 💡"Does this character's motivation feel consistent?"
  • 💡"Write a brief synopsis of the story so far."
AI Features

Continuity Checker

The Continuity Checker sends your entire script to Gemini and asks it to identify inconsistencies — the kind that slip through manual proofreading. Results are returned as tagged, categorized findings.

What it scans for

  • ⚠Character name inconsistencies ("JOHN" vs "JONATHAN")
  • ⚠Timeline jumps and anachronistic details
  • ⚠Location naming inconsistencies
  • ⚠Props or details referenced before they're introduced
  • ⚠Character knowledge inconsistencies
  • ⚠Wardrobe or appearance continuity issues

How to run it

  1. Open the Tools menu or click the shield icon in the toolbar.
  2. Click AI Continuity Check.
  3. The checker sends your full script and waits for a response (may take several seconds for long scripts).
  4. Findings are displayed as a list with category badges. Click a finding to jump to the relevant passage.
Long scripts Gemini's context window can handle feature-length screenplays comfortably. For very long novels, the checker may sample sections rather than read every word.

Export

Exporting Scripts

Screenplays and Stage Plays can be exported in three formats, each suited to a different use case.

.pdf

Print-ready, page-formatted output. Includes optional title page, scene numbers, page numbers, and revision watermark.

.docx

Microsoft Word format. Each script element maps to a correctly-styled Word paragraph with proper indentation and capitalization.

.fountain

Open screenplay markup format. Compatible with Highland 2, Fade In, Final Draft (via import), and other tools in the Fountain ecosystem.

Fountain format The .fountain export uses the open standard at fountain.io — plain text with simple markup that most professional screenplay tools can import. It is ideal for moving a draft into another application.
Export

Exporting Novels

Novels export in three formats. All formats include every chapter in order.

.pdf

Manuscript-formatted PDF. Optional table of contents with page numbers. Chapter headings are styled and numbered.

.docx

Word-compatible manuscript. Chapters are styled as Heading 1 sections with body text in a readable 12pt font.

.txt

Plain text export. Chapter titles are separated by double line breaks. Clean and portable — useful for pasting into other tools.

Table of contents Enable Table of contents in export settings before exporting to PDF. The TOC lists every chapter with the page it starts on and appears at the front of the document.
